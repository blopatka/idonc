Frequently Asked Questions

P: What should I do to register new user?
O: If You can read that page, it means that You are already registered user.

P: How can I read info about users.
O: You have to open page "Users->List", and then choose interesting person and click "detaild".

P: How can I edit info about my user.
O: You have to open page "Users->Edit" and fill given form.

P: How can I check list with active computation project?
O: You have to open page "Projects->List".

P: How can I join project?
O: On page "Projects->List" click "activate" link in chosen project.

P: How can I run computation client application?
O: Go to page "Projects->Run" and click the link on this page.

P: What requirements are for running client application.
O: Client should have Java version 6 installed on operating system.
